  • Aula 3: Estrutura básica de um arquivo HTML

    • (Voltar)

    Abrir a pasta do projeto e criar um arquivo chamado 'index.html', o nome 'index' é padão por significar o inicio

    Para alterar o nome na aba, é so alterar a tag TITLE no HEAD

    O HTML tem três partes, o HTML que é o arquivo como um todo, o HEAD que vai abrir o que estiver dentro dele antes de mostrar para o usuario e o BODY, que é onde está todo código

  • Aula 4: Falando sobre tags

    • (Voltar)

    O que estiver etre as tags o navegador ira intepretar e apresntar. Exemplos de tags:

    A tag 'i' é italico, o 'strong' é negrito, 'a' sãos as ancoras que servem para links

    Algumas tags nao tem o fechamento, como a 'input' porque é um bloco de escrita, e outra tambem é a 'img'

  • Aula 5: Atributos das tags

    • (Voltar)

    As tag's devem ser criadas da seguinte forma = < tag > < / tag >

    Atributos são propiedadas que uma tag pode ter, alguns são mais globais e outros mais especifica.

    Algumas propriedades globais: '#id' bastante usado pelo JavaScrpit, 'style' usado pelo CSS, 'class' para padronizar, 'blank' na ancora 'a' para abrir link em outra aba

    Algumas propriedades especificas: 'type[number, color,text]' do input; 'src' do img

  • Aula 6: Textos

    • (Voltar)

    Os 'h' são os titulos

    O 'p' são paragrafos

    O 'blockquote' serve para citação

    O 'u' é para underline

    O 'mark' serve como um marcador de texto

    O 'sup' como um 'expoente' no texto

    O 'small' criar texto pequeno

  • Aula 7: Listas ordenadas e listas desordenaas

    • (Voltar)

    'ol' é para lista ordenada e 'ul' é para lista nao ordenada , lembrando de colocar a tag 'li'

  • Aula 8: Outras informações

    • (Voltar)

    Usar o < hr /> para criar uma linha na horizontal

    Usar a ancoragem para linkar a algum local especifio dentro da propia página
